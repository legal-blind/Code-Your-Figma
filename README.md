Design Used:
For this assignment i decided to use the Balenciaga website as a reference point. The homepage has a clean layout with centered text, large images, and wide empty spaces. I recreated this style by keeping the background simple, using big images, and using minimal text. The two holiday images from the real site are included in the sections.

JavaScript Interactions:
1. Theme toggle changes the page between light and dark mode.
2. Email box checks for a valid email and shows a message under the button.
3. The images get a small scale-up effect when you hover over them.
